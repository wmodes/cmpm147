Experiment 1 - Hello Glitch/Generator

Translated from cmpm147-ex01-bad-quests on glitch.com

Description

Create a unique generator on glitch.com and move the working generator to my portfolio page.

Technical

A grammar is defined with simple replacement tokens. A replacer() function picks a random replacement for any token that appears in the text.

Reflection

I'm planning the class, so it toook a bit to get everything sorted and decide exactly how I wanted to do it. My only concern is that the transition from glitch.com to a static webpage might be a little rough for students.
